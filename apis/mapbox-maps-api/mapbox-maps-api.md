# Mapbox Maps API


The Mapbox Maps API reads and renders raster map tilesets and vector map tilesets.  Tilesets can be retrieved as images, TileJSON, or HTML slippy maps for embedding

**Docs / Website**: https://docs.mapbox.com/api/
